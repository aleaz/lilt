# Validation metadata

Holds **coverage indexes and notes** derived from [CLAIMS.md](../CLAIMS.md) and
asset READMEs.

## Rules

- Do **not** commit generated campaign dumps, LLM logs, or TM snapshots here.
- Optional future files (e.g. coverage summaries) must be human-maintained or
  regenerable; prefer updating `CLAIMS.md` Evidence column first.
- This directory is not a playground for scratch reports.

Empty until maintainers add an explicit coverage note file.
